# Estructura de un proyecto
```plaintext
/Proyecto_Ada
|
├── src/                   # Directorio para los archivos fuente (.ads y .adb)
│   ├── main.adb           # Archivo principal del programa
│   ├── paquete.ads        # Especificación del paquete
│   ├── paquete.adb        # Implementación del paquete
│   ├── utilidades.ads     # Especificación de utilidades
│   ├── utilidades.adb     # Implementación de utilidades
│
├── obj/                   # Directorio para los archivos objeto (.o) y binarios intermedios
│
├── bin/                   # Directorio para los ejecutables finales
│
├── proyecto.gpr           # Archivo de proyecto GNAT
│
└── README.md              # Descripción del proyecto
```
